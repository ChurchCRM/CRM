A free Church Web CRM system

With Church Web CRM you can manage your church congregation directory, Sunday School, Volunteers, and pledges.

Mobile Friendly

Once it is setup you can access it from anywhere using a PC's web browser, an iPad/tablet or a smart phone.

Always in development

This church CRM has been around for 10+ year, it started as InfoCentral, then ChurchInfo now ChurchCRM.io is on GitHub which mean a new generation of developers will always be adding features or creating new branches.

LAMP Stack

LAMP is the acronym for the combination of the operating system and software stack known as Linux, Apache, MySQL, and PHP.. Since Church CRM is a LAMP software it can be installed on most web hosting companies.

Integrations

  • Map of the Church congregation via Google Maps
  • Integration with MailChimp lists

Congregation Features

  • Organizes Church members into families
  • Upload family and members photo
  • Take family and members photo via phone/tablet/webcam

Sunday School

  • Create Groups
  • Sunday School class report with kids and parents info
  • Email Sunday School class, teachers, and parents

Reports

  • Email a family a PDF with the CRM info
  • Email every family at once requesting info verifcation

Docker

pull the docker registry image using

docker pull churchwebcrm/docker

or visit Church Web CRM Docker GitHub repo
